# Bakugo

## Contributing guide

### Set up

Install [`just`](https://github.com/casey/just#installation):
```bash
cargo install just
```

Then run set up:
```bash
just init
```

### Testing

Run on a specific file:
```bash
just run tests/examples/01_sanity.bakugo
```

Run tests (with review):
```bash
just test
```
